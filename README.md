# Mini Project

## Overview

This is a starter project skeleton for the internship mini project assignment. It is built with **Spring Boot 4.1.0** and **Java 21**.

## Project Structure

```
src/main/java/de/lhind/internship/mini/project/
├── Application.java          # Main entry point
├── exception/                # Global exception handling
```

## Notes

- The assignment details will be shared during the class.
- Make sure you can **build and run** the project before the session starts.
- Upload the project to your individual repository on GitHub and make sure you can **push changes** to it.
